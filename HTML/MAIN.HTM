Getting Started

To get started using this CD, you need to install the software package. Before you begin, however, make sure your computer meets the following system requirements.

Hardware requirements for Microsoft® Visual C++TM 6.0 Introductory Edition

  • PC with a Pentium-class processor, 90 MHz or higher
  • Microsoft Windows 95 or later or Microsoft Windows NT 4.0 with Service Pack 3 (or later)
  • Microsoft Internet Explorer 4.01 Service Pack 1 (included)
  • CD-ROM drive
  • VGA or higher-resolution monitor; Super VGA recommended
  • Microsoft Mouse or compatible pointing device
  • 24 MB RAM for Windows 95 or later (48 MB recommended)
  • 32 MB for Windows NT 4.0 or later (48 MB recommended)
  • Disk space required for basic install of VC++ only: 266 MB typical, 370 MB maximum
  • Disk Space required for VC++ and Service Pack install: 345 MB
    • (NOTE: Because the Service Pack mostly replaces files, it only adds about 30 MB typically)

(NOTE: Hardware Requirements for Visual Studio 6.0 Service Pack 3 are the same)

Installing the Software

Now that you have met the minimum requirements for the software installations, read the license agreement. Please read the entire license and make sure you agree with it. Now you are ready to install.

Microsoft® Visual C++TM 6.0 Introductory Edition

When you put this CD into your CD-ROM drive, an Autorun splash screen appeared. In order to install Visual C++, you need to go back to that splash screen.

Go back to the screen that popped up when you loaded the CD.

If you don't still have that splash screen on your desktop, eject the CD and reinsert it and the splash screen should reappear.

You will see a button for Microsoft® Visual C++TM 6.0 Introductory Edition. Click the button to begin downloading and installing the application. When done with the installation, reload this file in your browser.

Microsoft Service Pack

By default, Visual C++ 6.0 Introductory Edition is designed to run with Microsoft Internet Explorer 4.0. If you are running Microsoft Internet Explorer 5.0, you can download Microsoft Visual Studio 6.0 Service Pack 3 at http://msdn.microsoft.com/vstudio/sp/vs6sp3/default.asp to improve the interface.

(NOTE: Currently Microsoft does not offer the Service Pack for only Visual C++, so the entire 120 MB Visual Studio Service Pack has to be downloaded from the Web.)

Examples

Once you have installed the software, you are ready to begin working through the exercises from the book. Click here to load the directory listing for the examples.

This will bring you to a window that shows the examples directories of the CD. They are arranged by chapter number and contain the example code for each particular chapter. You can copy them to your hard drive and modify them as you work your way through the book.

You may also click directly on the Examples button in the window to your left, or on the startup screen. Or you may use the Windows Explorer to copy the examples directly from the CD to your hard drive.

Links

Click on the Links button (in the window to your left, or on the startup screen) to obtain more information from the Internet, download software updates from the manufacturers' sites, and join discussions on C++ with your peers.

For the most up-to-date code examples and live WWW links, visit http://www.deitel.com/products_and_services/publications/index.htm

NOTE: You must have a live Internet connection to use the links.
