B


bounding box
Defines the part of an image or geometry that is detectable or nontransparent (tangible). The ImageBvr.boundingBox method (DAImage.BoundingBox) returns a 2-D bounding box (a Bbox2Bvr or DABbox2 class object) that encloses all the tangible parts of the image. The GeometryBvr.boundingBox method (DAGeometry.BoundingBox) returns a 3-D bounding box (a Bbox3Bvr or DABbox3 class object) that encloses all the tangible parts of the geometry.
