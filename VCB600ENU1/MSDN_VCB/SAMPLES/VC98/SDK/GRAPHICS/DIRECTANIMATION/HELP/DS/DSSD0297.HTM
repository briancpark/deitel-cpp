Analog Video CrossBar


The WDM-based Analog Video Crossbar filter is capable of selecting or routing analog video. The single output stream represents a hardware path for analog baseband video. One of the input pins comes from a TV Tuner (the TV Tuner Filter). Other input pins support video streams.

The Analog Video Crossbar filter exposed the IAMCrossbar interface which is used for routine and source selection.

Note that this filter is available through the Windows 98 and Windows NT 5.
