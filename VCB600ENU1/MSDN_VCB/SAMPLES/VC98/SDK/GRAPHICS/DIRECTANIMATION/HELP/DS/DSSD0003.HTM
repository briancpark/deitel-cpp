What's New in the DirectShow SDK?


The Microsoft® DirectShow™ SDK (formerly the Microsoft ActiveMovie™ SDK) is now part of the Microsoft DirectX® Media SDK. Features added since ActiveMovie 1.0 include video and audio capture and compression, device enumeration, digital versatile disc (DVD) support, digital video (DV) support, device control and timecode support, and multimedia streaming. See the topics in the following lists for details of all the DirectShow SDK's new features.
