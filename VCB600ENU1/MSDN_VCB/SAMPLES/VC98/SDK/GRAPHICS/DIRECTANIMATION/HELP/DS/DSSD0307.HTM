Cutlist File Source


This source filter can output different pieces of different .avi or .wav files seamlessly and efficiently. It can be used for non-linear video editing and for audio editing.

Using the CutListGraphBuilderObject, the SimpleCutList object, and the VideoFileClip and AudioFileClip objects, an application can build a cutlist out of pieces of .avi and .wav, and use the DirectShow Cutlist Source Filter to play it.
